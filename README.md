# mmt

mmt is an arithmetic expression transpiler to MIPS assembly JFF.

## Build locally

```
cmake -S . -B ./build
cmake --build ./build

# Spawn a REPL
./build/src/mmt-exe

# Run the TUI
./build/ui/mmt-tui
```
